Express Checkout Demo

Separate Charges & Transfers with on_behalf_of

Option 1: Client-side Confirm Client-side →

The frontend creates a PaymentIntent upfront, collects payment via Express Checkout, and confirms the payment client-side. After success, it triggers a separate Transfer.

1. Create PaymentIntent → 2. Customer pays → 3. stripe.confirmPayment() → 4. Create Transfer

Option 2: 2-Step Confirmation Server-side →

The frontend collects payment details and creates a ConfirmationToken. The backend creates and confirms the PaymentIntent in one call, then creates the Transfer.

1. Express Checkout → 2. createConfirmationToken() → 3. Backend confirms + transfers
